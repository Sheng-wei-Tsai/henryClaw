## Constraints
- **不可對 `#1` 的任務內容做推測。** 目前沒有標題語意、正文、留言或可驗證背景，若硬補內容，容易製造錯誤長期記憶。

- **不確定性高。** 目前能寫進長期記憶的，多半是「追蹤入口存在」與「資訊缺口本身是阻塞」；除此之外都應保守處理。
