## Constraints
- **資料來源極度稀薄**：近 30 天可用資訊幾乎只來自 `#1`，且內容不足。

- **issue metadata 是目前主要可信來源**；未寫進 repo 的背景，即使可能存在，也不能當成穩定事實。
- **沒有可驗證的進度紀錄**：目前看不到明確實作、除錯、規劃里程碑或完成回報，因此不能把任何工作線視為已啟動或已推進。
